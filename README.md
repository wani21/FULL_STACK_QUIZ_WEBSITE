# FULL_STACK_QUIZ_WEBSITE
A Simple and easy to use quiz website for students and teacher , teacher can create quiz , student can attempt the quiz 


#steps to run 

1. cd Quiz_project
2. cd BACKEND
3. cd Quiz_backend
   npm install

   #4 Now start Frontend

   1. cd Quiz_project
   2. cd FRONTEND
   3. cd quiz
   4. ng serve
  
      project will be run on :-

      http://localhost:4200/
